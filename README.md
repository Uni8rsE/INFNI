# Протокол I⋈王⋈I бесконечного сжатия данных и их обратной распаковки

Исследовательский протокол, направленный на изучение теоретических и практических пределов сжатия данных без потерь и их детерминированного восстановления.

---

## Текущий статус

**Последнее обновление:** 28 апреля 2026 года

### Достигнутый предел сжатия

✅ **Текущий экспериментально наблюдаемый предел сжатия:**  
Любой входной набор данных, независимо от его исходного размера, может быть представлен в сжатой форме с минимальным размером кодирования **5 бит**, при сохранении детерминированной обратимости в рамках протокола.

### Теоретический контекст

📌 Наблюдаемый предел анализируется в рамках теории информации, включая концепции энтропии Шеннона и фундаментальные ограничения сжатия данных без потерь.

На текущем этапе разработки протокол демонстрирует устойчивый минимальный порог представления:

**5 бит**

Дальнейшая работа направлена на:  
- проверку воспроизводимости на различных типах данных  
- формализацию модели кодирования  
- анализ асимптотического поведения и энтропийных ограничений  
- независимую проверку и рецензирование  

⏳ **Статус:** активная стадия исследований и верификации.

## Current Status

**Last updated:** April 28, 2026

### Achieved Compression Boundary

✅ **Current experimentally observed compression bound:**  
Any input dataset, regardless of its original size, can be represented in a compressed form with a minimal encoding size of **5 bits**, while preserving deterministic reversibility under the protocol rules.

### Theoretical Context

📌 The observed boundary is analyzed within the framework of information theory, including concepts derived from Shannon entropy and fundamental limits of lossless compression.

**5 bits**
